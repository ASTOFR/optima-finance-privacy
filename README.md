# ClearBudget

**ClearBudget** est une application de gestion financière personnelle pour Windows, conçue pour vous offrir une vue claire sur votre argent avec un design premium, tout en garantissant une confidentialité totale.

## 🚀 Fonctionnalités Principales
- **100% Local et Sans Abonnement :** Vos données restent sur votre ordinateur. Pas de cloud, pas d'abonnement.
- **Sécurité de Grade Militaire :** Vos comptes sont protégés par un chiffrement AES-256-GCM.
- **Design Premium :** Interface moderne, claire et intuitive (Glassmorphism, Dark/Light Mode).
- **Conseiller IA Local :** Intégration possible avec Ollama pour des bilans financiers intelligents, générés directement sur votre machine.

## 🔒 Confidentialité ("Privacy by Design")
Nous avons conçu ClearBudget avec la conviction que vos finances ne regardent que vous. L'application n'inclut aucun tracker et n'envoie **aucune donnée** sur Internet.

## 💻 Installation
Téléchargez la dernière version depuis la section [Releases](https://github.com/votre-compte/clearbudget/releases) ou retrouvez l'application prochainement sur le **Microsoft Store**.
